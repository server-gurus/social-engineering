# social-engineering
Übersicht, wie plausible Fake-Rechnungen, Lieferantenidentitäten und Bankdatenmanipulationen im Umfeld eines digitalisierten Rechnungseingangs erkannt, verhindert und bearbeitet werden. Der Schwerpunkt liegt auf organisatorischen Massnahmen, technischen Kontrollen, Fraud-Detection-Werkzeugen und einem belastbaren SAP Fraud-Workflow.
